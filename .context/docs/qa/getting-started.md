# How do I set up and run this project?

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd AgenciaF3F

# Install dependencies
npm install
```

### Running

```bash
# Development
npm run dev
```
