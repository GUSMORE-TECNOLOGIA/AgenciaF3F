================================================================================
DECISÕES TÉCNICAS
================================================================================

1. Stack Tecnológica:
   - Frontend: React 18 + TypeScript + Vite
   - Estilização: Tailwind CSS
   - Backend: Supabase (PostgreSQL + Auth + Storage)
   - Roteamento: React Router DOM v6

2. Estrutura de Pastas:
   - components/: Componentes reutilizáveis
   - pages/: Páginas da aplicação
   - services/: Serviços (Supabase, APIs)
   - types/: Definições TypeScript
   - contexts/: Contextos React
   - utils/: Funções utilitárias

3. Links Úteis:
   - Cada cliente terá um objeto LinksUteis com 11 campos específicos
   - Armazenado como JSONB no Supabase
   - Interface dedicada para gerenciamento

4. Perfis de Usuário:
   - Cada usuário será responsável por um ou mais clientes
   - Sistema de roles: admin | user
